Detailed Description

Implements a file server providing a simple protocol. The file server has the following 3 commands (1) "NEXT", (2) "CHECKSUM", (3) "CONTENT".

If the server receives command "NEXT", it creates a new file content. If the server receives command "CHECKSUM", its response is the checksum of the current file content. If the server receives command "CONTENT", it sends the content of the file as response.
